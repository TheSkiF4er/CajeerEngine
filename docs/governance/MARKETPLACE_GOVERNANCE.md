# Правила governance для Marketplace

## Доверенные издатели (trusted publishers)
- уровни: community / verified / certified

## Правила пакетов
- подписи обязательны для verified/certified
- метаданные лицензии обязательны
- требования безопасности: без malware, без кражи учётных данных, без скрытого трекинга

## Применение правил (enforcement)
- автоматические сканы (foundation) + ручная модерация для featured‑пакетов
