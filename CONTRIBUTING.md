# CONTRIBUTING

Спасибо за вклад в CajeerEngine.

## Как предложить изменения
- Создайте fork
- Ветка: `feature/<короткое-имя>` или `fix/<короткое-имя>`
- Откройте Pull Request с описанием:
  - что меняется
  - почему
  - как тестировалось

## Стиль
- PHP 8.2+
- Имена классов: PascalCase, методы: camelCase
- Минимальные зависимости

## Тесты
- Размещайте тесты в `tests/`
- Добавляйте сценарии для шаблонизатора и роутера
